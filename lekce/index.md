# Python: From Zero to Hero

Kompletní kurz Pythonu v češtině — od absolutních základů po senior/architect úroveň.

**134 lekcí · 134 programů · 14 mini-projektů**

---

## Jak začít

```bash
# Instalace MkDocs (pro prohlížení kurzu)
pip install mkdocs-material

# Spuštění lokálního webu
mkdocs serve
# → otevři http://127.0.0.1:8000
```

```bash
# Spuštění libovolného programu
python3 programy/l01_uv_demo.py
python3 programy/l101_solid.py
```
