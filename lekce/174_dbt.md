# Lekce 174: dbt — data transformation

dbt (data build tool) transformuje data v datovém skladu pomocí SQL. Přidává verzování, testování, dokumentaci a lineage k SQL transformacím.

---

## 🚀 Instalace

```bash
uv add "dbt-core" "dbt-postgres"   # nebo dbt-duckdb, dbt-bigquery
dbt --version
```

---

## 📁 Struktura projektu

```
muj_dbt_projekt/
├── dbt_project.yml        ← konfigurace projektu
├── profiles.yml           ← přihlašovací údaje (mimo git!)
├── models/
│   ├── staging/           ← suroviny z datových zdrojů
```

## ⚙️ Konfigurace

`dbt_project.yml`:
```yaml
name: muj_projekt
version: "1.0.0"
profile: muj_profil

models:
  muj_projekt:
    staging:
      +materialized: view      # views (levné, vždy aktuální)
    intermediate:
      +materialized: ephemeral # CTEs (vložené, žádná tabulka)
    marts:
      +materialized: table     # tabulky (rychlé čtení)
      +schema: analytics
```

`profiles.yml`:
```yaml
muj_profil:
  target: dev

      schema: dbt_dev
```

## 📝 Models — SQL transformace

`models/staging/stg_orders.sql`:
```sql
-- Staging: jen rename + cast, bez logiky
with source as (
    select * from {{ source('raw', 'orders') }}
),

renamed as (
    select
        id                          as order_id,
        customer_id,
        cast(created_at as date)    as order_date,
        status,
        round(total_amount, 2)      as total_amount_czk,
        -- Metadata
        _loaded_at                  as loaded_at
    from source
    where _deleted = false          -- soft deletes
)

select * from renamed
```

`models/marts/orders.sql`:
```sql
-- Finální mart: objednávky s obohacenými daty
with orders as (
    select * from {{ ref('stg_orders') }}    -- ref() = závislost
),

customers as (
    select * from {{ ref('stg_customers') }}
),

final as (
    select
        o.order_id,
        o.order_date,
        o.total_amount_czk,
        o.status,
        c.customer_name,
        c.segment,
        -- Vypočítané metriky
        case
            when o.total_amount_czk >= 10000 then 'high_value'
            when o.total_amount_czk >= 1000  then 'medium_value'
            else 'low_value'
        end as order_tier,
        datediff('day', c.first_order_date, o.order_date) as days_since_first_order
    from orders o
    left join customers c using (customer_id)
)

select * from final
```

## 🧪 Testování

`models/marts/schema.yml`:
```yaml
version: 2

models:
  - name: orders
    description: "Finální tabulka objednávek"
    columns:
      - name: order_id
        description: "Unikátní ID objednávky"
        tests:
          - unique
          - not_null
      - name: status
        tests:
          - accepted_values:
              values: ['pending', 'processing', 'shipped', 'delivered', 'cancelled']
      - name: customer_id
        tests:
          - not_null
          - relationships:
              to: ref('customers')
              field: customer_id
      - name: total_amount_czk
        tests:
          - not_null
          - dbt_utils.expression_is_true:
              expression: ">= 0"
```

Vlastní test (`tests/assert_orders_not_negative.sql`):
```sql
-- Selže pokud vrátí řádky (assertion test)
select order_id
from {{ ref('orders') }}
where total_amount_czk < 0
```

## 🔧 Makra (DRY SQL)

`macros/safe_divide.sql`:
```sql
{% macro safe_divide(numerator, denominator) %}
    case
        when {{ denominator }} = 0 or {{ denominator }} is null
        then null
        else {{ numerator }} / {{ denominator }}
    end
{% endmacro %}
```

Použití v modelu:
```sql
select
    customer_id,
    total_orders,
    total_revenue,
    {{ safe_divide('total_revenue', 'total_orders') }} as avg_order_value
from customer_summary
```

## ⚡ Spuštění

```bash
# Build (transformuj a testuj)
dbt run                      # všechny modely
dbt run --select stg_orders  # jeden model
dbt run --select marts.*     # celá složka
dbt run --select +orders     # orders + všechny závislosti

# Testování
dbt test
dbt test --select orders

# Dokumentace
dbt docs generate
dbt docs serve               # http://localhost:8080

# Celý pipeline
dbt build                    # run + test + seed + snapshot
```

## 🎯 dbt + Python modely (dbt 1.3+)

```python
# models/python/ml_predictions.py
import pandas as pd
from sklearn.ensemble import RandomForestClassifier

def model(dbt, session):
    # Načti dbt model jako DataFrame
    df = dbt.ref("features").to_pandas()

    X = df.drop(columns=["target", "customer_id"])
    y = df["target"]

    clf = RandomForestClassifier(n_estimators=100)
    clf.fit(X, y)

    df["prediction"] = clf.predict(X)
    df["probability"] = clf.predict_proba(X)[:, 1]

    return df[["customer_id", "prediction", "probability"]]
```
